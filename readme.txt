== Installation ==

The “Adsense for AMP” plugin requires “AMP” plugin for WordPress by “Automattic”.  So if you haven’t installed AMP plugin yet, kindly follow these steps:

1. Log in to your WordPress admin and navigate to Plugins > Add new.
2. Inside “Search Plugins” field, type “AMP”. From the results, install the AMP plugin as found in this URL: https://wordpress.org/plugins/amp/
3. After installation, activate the plugin via Plugins > Installed Plugins > AMP

Once that is done, to install the “Adsense for AMP” plugin, follow these steps:

1. Log in to your WordPress admin and navigate to Plugins > Add new.
2. Inside “Search Plugins” field, type “Adsense for AMP”.
3. After installation, activate the plugin via Plugins > Installed Plugins > Adsense for AMP
Once the plugin is activated, you can navigate to the settings page for “Adsense for AMP”.


== Frequently Asked Questions ==

= How to test the ads setting? =
To test the changes, save the settings and view any blog post on a mobile device by adding "/amp/" at the end of the url, eg: http://yourdomain.com/your-sample-article/amp/
Note: Don't see any changes? try to clear the site/browser cache and refresh.
